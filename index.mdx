## Get up and running

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/samuelitoo01/Tic-Tac-Toe-Game.git
    cd Tic-Tac-Toe-Game
    ```
  </Step>
  <Step title="Open the game">
    Double-click `index.html` to open it in your browser, or serve it with a local server:
    ```bash
    python -m http.server 8000
    ```
  </Step>
  <Step title="Start playing">
    Player X goes first. Click any empty cell to place your marker. Players alternate turns until someone wins or the board is full.
  </Step>
</Steps>

<Tip>
  No internet connection required. Once you have the files, the game works fully offline.
</Tip>
